## Getting started

Clone repo:

```
$ git clone https://github.com/paulohlips/beTheHero
```

Inside of projects folder, install all dependencies:

```
$ cd frontend
```

```
$ yarn (or npm install)
```

To run the front-end:

```
$ yarn start (or npm start)
```

Project will be running on http://localhost:3000
